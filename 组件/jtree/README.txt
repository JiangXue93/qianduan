使用方式：

1. 需要使用该插件的页面引入：（1）jquery.js（2）该目录下的./js/jtree.js文件

2. 调用方式参看index.html（插件容器使用jquery的id选择器）

3. 返回值为操作后的数据



说明：
1.数据输入：必须有一个根节点，实际操作时可将文本（text）设置为“全选”
	
	格式： [
		{
			id: 0,//数字id
            		text: "食材",//页面显示字样
            		state: {
                		checked: false,//选择状态（全选状态）
                		childCheck: true,//半选状态（孙子元素被选状态）
                		open: true,//是否处于展开状态
                		wrap: true,//是否有子节点，可展开
                		search: true,//查询命中状态(【重要】初始化为true便于第一次渲染)
            			},
            		nodes: [{...同样格式数据..},//子元素
				{...同样格式数据..}	
			]
		 }	
		]

2.css样式说明：由于没有设计稿，该部分只完成基本功能操作，样式更改可参照index.html
